# walkSafe
“WalkSafe is a concept mobile app designed in Figma, focusing on improving pedestrian safety and navigation"

## Features
- Optimized walking routes for pedestrians
- Report unsafe areas in real-time
- Request new walking paths
- User-friendly and accessible UI/UX
- Real-time weather display to enhance walking safety and planning
